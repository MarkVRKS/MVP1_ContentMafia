Открытые позиции

Senior Full-Stack Developer

Удалённо • Full-time

👨‍💻

Разработка и поддержка наших SaaS-продуктов. Работа с React, Node.js, PostgreSQL, AI API. Создавай инструменты, которыми пользуются тысячи пользователей.

Требования:

  • Опыт 3+ года в full-stack разработке
  • Знание React, Node.js, TypeScript
  • Опыт работы с AI API (OpenAI, Anthropic)
  • Понимание DevOps практик

Growth Marketing Specialist

Удалённо • Full-time

📊

Планирование и проведение маркетинговых операций для клиентов. Работа с данными, A/B тестирование, оптимизация конверсий и масштабирование успешных кампаний.

Требования:

  • Опыт 2+ года в performance-маркетинге
  • Знание аналитики (GA4, Amplitude, Mixpanel)
  • Опыт работы с рекламными платформами
  • Data-driven подход к принятию решений

Content Strategist

Удалённо • Full-time

🎨

Создание контент-стратегий для клиентов, генерация вирусных идей, работа с AI-инструментами для оптимизации контента. Создавай контент, который работает.

Требования:

  • Опыт 2+ года в контент-маркетинге
  • Портфолио успешных кампаний
  • Знание трендов и платформ (TikTok, Instagram, YouTube)
  • Опыт работы с AI-инструментами (ChatGPT, Midjourney)

AI Engineer

Удалённо • Full-time

🤖

Интеграция AI в наши продукты, разработка промптов, fine-tuning моделей, работа с LLM API. Будь на переднем крае AI-технологий.

Требования:

  • Опыт работы с LLM API (OpenAI, Anthropic, etc.)
  • Знание Python, prompt engineering
  • Понимание ML/AI концепций
  • Опыт с RAG, embeddings, vector databases
